# Winter_Algorithm
 그냥 겨울방학 알고리즘 공부

# 개요
 겨울방학을 시작해서 알고리즘 기초를 복습하고 문제해결 기법을 기르는 시간과 간단한 프로젝트를 한 것이 이번 방학의 목표이다.
